# ImageGenApp

A simple C# console application that generates images using OpenAI's DALLÂ·E API. It takes a prompt as input, sends it to the OpenAI API, and saves the generated image locally.

## Features

- Generate images based on text prompts using OpenAI's image generation API.
- Save generated images as PNG files.
- Supports API Key authentication using an environment variable.

## Requirements

- .NET 6.0 or later
- An OpenAI API Key (required to access the OpenAI Image Generation API).
